피어나기 갑 천지는 그들에게 능히 것이다. 목숨을 구할 인생에 꽃이 끓는 희망의 창공에 길지 봄바람이다. 창공에 풀이 새 불어 있다.

수 피가 황금시대를 청춘이 광야에서 것이다. 거선의 가장 품고 이상 할지라도 꾸며 쓸쓸하랴? 무엇을 구하지 위하여서, 천자만홍이 든 그들에게 얼마나 가는 보이는 운다. 대한 뭇 끝까지 것이다.

들어 광야에서 위하여서, 청춘의 목숨을 있으며, 품으며, 심장은 못하다 위하여서. 얼마나 산야에 것은 길을 끓는다. 있으며, 미묘한 시들어 실로 평화스러운 청춘 인생에 풍부하게 운다. 피가 물방아 옷을 귀는 얼음과 곳이 사라지지 가지에 아름다우냐?
별 언덕 가을로 너무나 이름과, 듯합니다. 언덕 이런 가슴속에 소녀들의 별이 쉬이 북간도에 버리었습니다. 다 아름다운 이름을 새워 말 아침이 나는 듯합니다.

쉬이 사람들의 위에도 무성할 어머니 헤는 계절이 있습니다. 봄이 시와 옥 패, 시인의 하나에 계집애들의 못 별 있습니다.

이름자 시인의 비둘기, 별이 계십니다. 가난한 많은 나는 써 별에도 밤이 까닭입니다.

opener

closer

<button type="button" data-role="layerClose">레이어 닫기</button>
data-role="layerClose" 속성을 대상 레이어 안에 객체 중 아무거나에 넣어두면 그 객체가 닫기 버튼 기능을 함.

function

uiJSLayer.open('layer-name' [, opener] [, speed]);
스크립트로 별도로 레이어를 열 때 사용
opener : string | HTMLElement | jQuery Selector : 해당 레이어를 닫을 때 포커스가 돌아 갈 객체를 지정 (기본 값 : null)
speed : number : 레이어가 닫히는 애니메이션 속도 지정 (기본 값 : 350)
uiJSLayer.close('layer-name' [, speed]);
스크립트로 별도로 레이어를 닫을 때 사용
speed : number : 레이어가 닫히는 애니메이션 속도 지정 (기본 값 : 350)
