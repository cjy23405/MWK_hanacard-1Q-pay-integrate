기존에 소지하셨던 카드를 [분실/훼손/종류변경] 등의 사유로 교체 또는 추가 발급을 받으시는 경우 『카드 비밀번호』는 다음과 같이 적용됩니다.

카드 교체/추가 발급 시 비밀번호 운영기준

테이블이름
구분 내용
교체 발급
  1. 최근 6개월 유실적 카드 중 가장 최근 발급한 카드의 비밀번호
  2. ①이 없을 경우, 사용등록이 된 카드 중 가장 최근 발급받은 카드의 비밀번호
    • 단, 가족카드는 비밀번호 승계 우선순서 대상카드(①, ②)에서 제외
추가 발급 이용카드가 없을 시 가장 최근에 발급한 카드와 동일한 비밀번호 사용
단, 기존 비밀번호가 없을 시, 카드 발급 시 ARS를 통해 비밀번호 등록
