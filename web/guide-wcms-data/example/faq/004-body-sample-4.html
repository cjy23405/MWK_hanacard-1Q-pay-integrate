자주 묻는 질문

해외 이용 시(해외사이트 거래 포함)미화(USD)기준 거래 금액에 접수일의 하나은행 최초고시 전신환 매도율을 적용한 후, 국제브랜드사가 부과하는 국제브랜드수수료와 하나카드가 부과하는 해외서비스수수료를 포함하여 청구됩니다.
(단, 체크카드의 경우 1회 승인금액 100만원 이상의 거래(단, T&E업종* 제외), 또는 카드사가 정한 특정 브랜드(BC/Maestro/Amex Globalpay 등) 및 상품의 카드이용대금은 미화(USD) 기준 거래 금액에 카드승인시점의 하나은행이 고시한 전신환매도율을 적용한 원화금액으로 회원의 계좌잔액에서 즉시 인출됩니다.)

  • T&E 업종 : Travel and Entertainment 업종, 여행과 여가 관련 업종(항공, 호텔, 숙박, 렌터카 등)

해외 이용시 청구금액 산출방법

해외 이용 시 청구금액 = (거래미화금액 X 전신환 매도율¹) + 국제브랜드수수료² + 해외서비스수수료³

  1. 전신환 매도율 : 접수일의 하나은행 최초고시 전신환 매도율
  2. 국제브랜드수수료 = (거래미화금액 X 국제브랜드 이용수수료율(VISA, MASTER, JCB : 1%, Amex : 1.4%, UPI : 0.6%)) X 전신환 매도율
  3. 국제브랜드수수료 = (거래미화금액 X 국제브랜드 이용수수료율(VISA, MASTER, JCB : 1%, Amex : 1.4%, UPI : 0.6%)) X 전신환 매도율
  • 단기카드대출(현금서비스) 거래에 대해서는 청구금액에 단기카드대출(현금서비스) 수수료가 별도로 포함 됩니다.
    • 단기카드대출(현금서비스) 수수료 = 청구금액 x 단기카드대출(현금서비스) 수수료율 x 신용공여기간 ÷ 365
  • 해외현금인출 거래에 대해서는 $3 수수료가 별도로 포함됩니다.(해외서비스수수료는 부과 안됨)
