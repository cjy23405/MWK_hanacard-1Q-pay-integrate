      해외 이용시 청구금액 산출방법
      
	
      해외 이용 시 청구금액 = (거래미화금액 X 전신환 매도율¹) + 국제브랜드수수료² + 해외서비스수수료³
      
	
        
		
      1. 
			
        전신환 매도율 : 접수일의 하나은행 최초고시 전신환 매도율
        
		
        2. 
		
      2. 
			
        국제브랜드수수료 = (거래미화금액 X 국제브랜드 이용수수료율(VISA, MASTER, JCB : 1%, Amex : 1.4%, UPI : 0.6%)) X 전신환 매도율
